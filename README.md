# Automated Cartoon Animation Generator

This project allows you to create simple animated stories automatically.

## Prerequisites

You need to have Node.js installed on your computer.

## Installation

1. Open your terminal.
2. Navigate to the project folder.
3. Run the following command to install the necessary tools:

npm install

## Database Setup

This project uses a database to save your stories. Run this command to set it up:

npm run db:push

## How to Run

To start the application, run:

npm run dev

Then open your web browser and go to: http://localhost:3000

## How to Use

1. Click on Create Animation or go to the Create page.
2. Enter a name for your project.
3. Write a short story. For example: The cat sat. The dog ran.
4. Click Generate Animation.
5. Wait for the animation to be created.
6. Watch your animation!

## Features

- Automatic story parsing
- Character and scene generation
- Audio narration
- Animated movements
